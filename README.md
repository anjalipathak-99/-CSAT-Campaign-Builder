# FeedbackFlow | CSAT Campaign Builder

A modern, full-featured Customer Satisfaction (CSAT) Campaign Builder dashboard. This single-page application allows product managers and marketers to design, brand, and preview high-converting customer feedback widgets in real-time using an interactive, sandboxed smartphone simulator.

## 🛠️ Tech Stack & Architecture

* **Frontend Library:** React 18 (Functional Components, Hooks)
* **Build Tool:** Vite (Ultra-fast Hot Module Replacement)
* **Styling Framework:** Tailwind CSS v3 (Utility-first responsive layout styling)
* **State Management:** React Context API (Centralized data store preventing prop-drilling)
